What We Do

School Of Freelancing is world first Linux based freelancing learning platform to provide one-to-one hands-on Linux freelancing training. After year of success a professional Linux freelancer name Masum funded School Of Freelancing and registered this domain on 2015-02-02 for help any newbie freelancer to grow up as a giant Linux freelancer, outsourcer, Linux mentor and high tech business entrepreneur. After successful complete our online ONE 2 ONE personal hands-on training a new trainee will enable to start earn by doing Linux freelancing from home.

Trainee also grow up from scratch:

  2. - Terminal command line using Ubuntu Linux
  3. - Bash shell scripting using Ubuntu Linux
  4. - Linux security and Linux server hardening
  5. - Linux Server Administration using DigitalOcean, AWS, Rackspace, Linode & Others VPS
  6. - Open-Source Software development using GitHub, GitLab & Bitbucket
  7. - Linux based Freelancing Using Freelancer, Fiverr, Peopleperhour & Guru dot com
